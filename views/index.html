API Project: Bus Ticket Booking Microservice

Available endpoints

  1. To book a ticket POST [project_url]/api/create
  2. To update ticket status and user details PUT [project_url]/api/ticket/:ticketId
  3. To view ticket status based on ticket_id GET [project_url]/api/ticket/:ticketId
  4. To get the tickets which are already bookedGET [project_url]/api/tickets/open
  5. To get the tickets which are not booked GET [project_url]/api/tickets/close
  6. To get the person details with the ticketIdGET [project_url]/api/user/:ticketId
  7. To signup to the token for admin ticket apiPOST [project_url]/private/admin/signup
  8. To login to the token for admin ticket api POST [project_url]/private/admin/logint
  9. To reset all ticket data as an admin POST [project_url]/private/tickets/reset
